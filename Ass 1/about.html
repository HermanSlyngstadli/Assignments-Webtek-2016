Questions:

  1. What does the server in the client server architecture do?
    The server is where the website is stored. It responds to client requests and sends the appropriate data back.
  2. What does the client do?
    The client is where the information is displayed to the user (often via a web browser). User input is transmitted to the web-server from the client.
  3. What is an IP address?
    An IP address is like the phone number of the website. A unique ID connected only to one single web page.
  4. What is DNS?
    DNS is a systematic storage system for IP addresses. It consists of often-used IP addresses cached in the server, as well as it can get other IP addresses from other DNS servers.
  5. What is the difference between the Internet and WWW?
    The internet is the whole network of computers all over the world connected via the TCP/IP protocol. The WWW is the information space you access via URLs in your browser to get documents and files. The WWW uses the internet.
  6. What is HTML?
    HTML is a markup language for displaying (structuring) information appealing to humans on screen. It is the standard for building web pages with stylization from CSS and JS.
